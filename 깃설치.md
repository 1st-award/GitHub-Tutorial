## 깃허브에 파일을 올릴 때 올라갈 이름과 이메일을 설정합니다.
  ```
  git config --global user.name "이름(닉네임)" (따옴표 포함)
  ```
  
  ```
 git config --global user.email "이메일" (따옴표 포함)
  ```
